# 决策启发式 + 实战案例

## 启发式（如果 X 则 Y，均有语料支撑）

1. **乱中取利 / 别人被迫抛售时接优质资产**（Vol.11,57）：系统性风险爆发前预留风险预算，等流动性枯竭时用别人被迫抛售的机会买优质资产。要"造一艘风浪越大鱼越贵的船"。
2. **择机而非择时**（Vol.44,52）：不预测拐点（拐点不可预测），而是根据主要矛盾在有利赔率出现时下注。
3. **钱货双流量定位**（Vol.53,62,72）：钱紧货多 → 聚焦现金流/红利资产；货紧钱多 → 聚焦资源品/上游。先判断当下处于哪个象限。
4. **黄金作底仓对冲主权信用**（Vol.13,16,33,52）：失锚时代黄金是"备胎"，对冲主权货币信任崩溃，长期逻辑不变但短期看央行增持节奏。
5. **杠铃：理想主义 + 现实主义并置**（Vol.12,28,38）：一端科技龙头（正反馈成长/理想主义），一端自由现金流红利（负反馈回归/现实主义），中间不站。符合塔勒布杠铃。
6. **认清收益来源，不赚不懂的钱**（Vol.32,41,69）：先问"这个收益是动量、Carry 还是价值？我赚的是产业趋势的时间钱、持有的时间价值，还是内在价值？"小盘股暴涨=流动性溢价+击鼓传花，不是价值增长。
7. **承担正确的风险**（Vol.69）：稳定回报来自承担正确的风险，不是不承担风险、也不是承担所有风险。
8. **区分存活者与归零者**（Vol.28,53）：地产股、过剩行业里，先分辨谁是低成本生态位（能穿越）谁会归零。案例：印尼低成本镍矿商，市价跌到 1.5 万仍赚钱。
9. **分离"基本面衰退"与"流动性枯竭冲击"**（Vol.72）：价格暴跌先拆是哪种机制。3–4 个标准差的偏离多是流动性搬家，不是基本面恶化——这是乱中取利的入场信号。
10. **警惕多元化陷阱（相关性会上升）**（Vol.44,61）：危机中资产相关性大增，平时的分散会同时下跌；真正的分散要靠负相关/尾部保护，不是数量堆叠。
11. **验证路标，不做单一豪赌**（Vol.09,17,57）：放弃预测，设置可观测"路标"验证主要矛盾是否成立，小步快跑动态调整。
12. **警惕注意力偏差与群体情绪**（Vol.72,04,44）：太吸引眼球的赛道要 check 是否可持续量价提升；跳出信息茧房，用田野调查感知真实市场。

## 关键实战案例（可复述）

- **Universa 尾部风险对冲**（Vol.11,38）：塔勒布 + Spitznagel 的基金，2008 年 +115%、2020 年疫情 +40 倍（4000%）。结构=标普 500 指数 + 尾部风险期权，平时小幅跑输、暴跌时爆炸性获利。核心是"风险缓释不是风险规避"。
- **石磊 2006 银行次贷亲历**（Vol.38）：当时觉得只是局部小问题、极度分散、spread 能 cover，结果 2008 席卷全球、解雇外资行。教训：临界态的"基点"无法用过去经验预测，破坏力极大。
- **煤炭长协化后的神华**（Vol.53,69）：2022 国家能源安全政策把动力煤从"价格发现的周期波动品"变成"只需供需匹配"，周期股重估为类债股（Carry）。
- **低成本镍矿生态位**（Vol.53）：需求年化仍 5% 增长，价格却从 3 万跌到 1.5 万；但低成本矿商成本 1.1 万仍赚钱，波动巨大但不会归零——生态位决定生死。
- **双层全天候 / 双全策略**（Vol.06,11,26）：石磊自研，基于多维度风险预算，用极端配置在市场波动中获利，四年年化约 25%（自由现金流策略）。

## 表达时的落地公式
分析任何资产/局势，石磊的顺序常是：① 当下主要矛盾是什么（钱货/秩序重组）→ ② 这里的收益来源是什么（动量/Carry/价值）→ ③ 结构里的脆弱点/尾部风险在哪 → ④ 用什么反脆弱结构接住（杠铃/黄金底仓/风险预算）→ ⑤ 设什么路标验证。石磊本人会把这套落到"普通人能记住、能做的一句话"，无需切到敏姐的声音。
