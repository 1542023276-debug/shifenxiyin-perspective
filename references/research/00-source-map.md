# 调研来源地图（本地语料模式）

**语料**：《十分吸引》播客逐字稿 Vol.01–Vol.72（2024-09-25 ~ 2026-06-28），约 245 万字符。
**主播**：石磊（理论内核/投资框架，本 Skill 交付的视角）、敏姐（刘敏，节目主持/提问者与概念落地者）。出品：吸引子。本 Skill 默认只呈现石磊的观点，敏姐仅在被明确问及「敏姐怎么看」时补充。
**模式**：纯本地一手语料。72 期逐字稿 = 最高权重一手来源，无网络搜索。
**调研方法**：
1. 抽取全部 72 期「节目简介」（41k 字）→ 构建主题地图与高频概念统计（≥3 次出现 = 真信念）。
2. 深读框架期（Vol.08/26/40/64）与实战期（Vol.11/38/53/72）的主播长段独白 → 提炼表达 DNA 与决策启发式。

## 内在张力（保留，不调和）
- **成长（理想主义）vs 价值（现实主义）**：正反馈 vs 负反馈，两种价值观内在冲突（Vol.12 杠铃组合并置，Vol.72 明说"成长与价值两种价值观内在冲突"）。
- **系统给不出预测（无用之用）vs 仍要做资产配置决策**：放弃预测拐点，却要"验证路标确认主要矛盾"下注。
- **宏大叙事 vs 微观路径**：爱讲百年史/文明/哲学，又强调"宏大叙事与投资之间隔着微观路径的千山万水"（Vol.25）。
